# 📱 Calculadora iOS

Una aplicación de calculadora desarrollada en Xcode con Swift. Una interfaz limpia e intuitiva para realizar operaciones matemáticas básicas.

## ✨ Características

- ➕ Operaciones básicas (suma, resta, multiplicación, división)
- 🎯 Interfaz limpia y fácil de usar
- 📱 Diseño responsivo para diferentes tamaños de pantalla
- ⚡ Cálculos rápidos y precisos

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Swift
- **IDE**: Xcode
- **Framework**: UIKit
- **Plataforma**: iOS

## 📋 Requisitos

- iOS 14.0 o superior
- Xcode 12.0 o superior
- MacOS 10.15 o superior

## 🚀 Cómo Usar

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/NaylanChamorroEspinoza/calculadora-app.git
   cd calculadora-app
   ```

2. **Abre en Xcode**
   - Doble clic en el archivo `.xcodeproj`
   - O abre Xcode y ve a File → Open

3. **Ejecuta la aplicación**
   - Presiona `Cmd + R` o el botón Play en Xcode
   - Selecciona un simulador o dispositivo físico
